# Glass UI and Dock for Google
Google の UI を「ガラス（半透明）風」にリデザインし、よく使う機能へ素早くアクセスできるドックを提供するユーザースタイル / 拡張機能向けプロジェクトです。クリーンでモダン、視認性を保ちながら控えめな光沢とぼかしを加えることで、Googleをより使いやすく、見た目も美しくします。

---

## 主要な特徴
- ガラス（Frosted Glass）風の UI：背景のぼかしと半透明レイヤーで洗練された見た目を実現
- ドック（ショートカットバー）：よく使うサービスへのクイックアクセスを提供
