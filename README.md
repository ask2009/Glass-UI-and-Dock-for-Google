# Glass UI and Dock for Google

ブラウザをiOS26のような「ガラス（半透明）風」のデザインに変更し、よく使うサイトなどに素早くアクセスできるショートカットバー（ピン留め機能）を追加します。
## 主な特徴
・UIがiOS26のような、半透明ガラス風デザインになります。
・UIに合ったショートカットバー（ピン留め機能）を追加します。

## その他
・Apple HIGのデザインを参考にしています。
・React ariaを使用し、react ariaの公式サイトを参考にしています。
